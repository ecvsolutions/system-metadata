eCyber là hệ thống quản lý phòng máy, áp dụng công nghệ Hybrid Data Management, cho phép các Desktop Client nội bộ ưu tiên làm việc với Cloud Server để tối ưu hiệu suất và khả năng mở rộng, đồng thời luôn duy trì kết nối với Local Gateway để hỗ trợ xác thực và đảm bảo tính sẵn sàng.

Trong trường hợp mất kết nối hoặc gặp sự cố với cloud, hệ thống lập tức chuyển sang xử lý hoàn toàn qua Local Gateway. Nhờ nền tảng message broker hiện đại, dữ liệu được bảo toàn và đồng bộ chặt chẽ giữa Local Gateway và Cloud Server sau khi kết nối được phục hồi, đảm bảo tính liên tục và tin cậy cho toàn bộ hệ thống quản lý cyber.
